# CRUD en PHP

Proyecto CRUD (Crear, Leer, Actualizar y Eliminar) desarrollado en PHP con conexión a MySQL.

## 📌 Tecnologías utilizadas

- PHP
- MySQL
- HTML5
- CSS3
- XAMPP
- Git & GitHub
